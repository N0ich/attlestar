Fleet --------------------------------------->
The Fleet class is the class designed to call
all the necessary classes to create a full
fleet for each players.
It contains attributes regarding the player
and the size of the fleet, and creates spawns,
as well as ships and weapons for every ships.
Its attributes are not really useful besides 
creating the fleet, but its method getShip();
parses all the necessary info in an array for
the whole fleet. Each ship will be in the
array. Its weapon are contained in the array
aswell.
